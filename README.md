# Software Download Website

## 📌 Project Description

This project is a Software Download Website that allows users to search, view, and download different types of software easily. It provides software details such as version, size, description, and system requirements.

## 🚀 Features

* Search software by name
* Categories (Windows, Android, Games, Utilities, etc.)
* Software description and details
* Download button with links
* User-friendly interface
* Responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

* index.html – Homepage
* style.css – Styling
* script.js – Functionality

## ⚙️ How to Run

1. Download or clone this repository
2. Open the project folder
3. Run `index.html` in your browser

## 🔐 Security

* Files are checked before uploading
* Basic validation for safe downloads
